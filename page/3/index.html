centos7安装Alfresco

发表于 2018-09-17   |   分类于 Alfresco   |  

centos7 install Alfresco

下载地址

  • Alfresco

设置可执行权限

chmod +x alfresco-community-installer-201707-linux-x64.bin
./alfresco-community-installer-201707-linux-x64.bin

配置/etc/hosts

127.0.0.1              {hostname.output}
{ipaddress.output}     {hostname.output}

启动

./alfresco.sh start

访问

http://ip:8080/share
