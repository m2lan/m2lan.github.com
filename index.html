在分类表中，通过左和右值来代表我们的嵌套节点。

  • 创建表并添加测试数据

    CREATE TABLE category (
    category_id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(20) NOT NULL,
    lft INT NOT NULL,
    rgt INT NOT NULL
) ENGINE=InnoDB DEFAULT CHARSET=utf8;
