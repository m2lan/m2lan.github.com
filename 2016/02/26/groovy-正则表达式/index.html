Groovy-正则表达式

元字符

  • .匹配任意单个字符 如：’hello’=~’hel.o’
  • ^匹配行的开始部分 如：’hello’=~’^he’
  • $匹配行的结束部分 如：’hello’=~’llo$’
  • 表示号前面字符或正则表达式出现0次或多次 如’hello’=~’l*’
  • +表示+号前面字符或正则表达式出现0次或多次 如’hello’=~’l+’
  • ?表示?号前面字符或正则表达式出现0次或多次 如’hello’=~’l?’
  • []匹配[]中字符集任一字符 如：’hello999’=~’[a-z0-9]*9’
  • {}限定次数，{1}表示限定一次、{1, 3}表示最少一次，最多3次 如’hello999’=~’hel{2}o9{1,3}’
  • \转义符 如’$’==~’\$’
  • |选择符 如’hello’=~’h(a|b|c|d|e)llo’
  • ()将封装的表达式组合起来 如’hello’=~’he(ll)o’

辅助符号

  • \d数字，等价于[0-9] 如’11223’=~’\d’
  • \D非数字，等价于[^0-9] 如’sfsf&((s’=~’\D’
  • \wWord字符，等价于[a-zA-Z0-9] 如’wres11’=~’\w’
  • \W非Word字符，等价于[^a-zA-Z0-9] 如’wres11*‘=~’\W’
  • \s空白字符，等价于[\t\n\f\r\v] 如’ ‘=~’\s’

  • \S非空白字符，等价于[^\t\n\f\r\v] 如’aaa’=~’\S’

    assert 'hello'=~'he'
assert 'hello'==~'hello'
assert 'hello'=~'l+'
assert 'hello'=~'l?'
assert 'hello999'=~'[a-z0-9]*9'
assert 'hello999'=~'hel{2}o9{1,3}'
assert '$'==~'\\$'
assert 'hello'=~'h(a|b|c|d|e)llo'
assert 'hello'=~'he(ll)o'
assert '11223'=~'\\d'
assert 'sfsf&((s'=~'\\D'
assert 'wres11'=~'\\w'
assert 'wres11***'=~'\\W'
assert '   '=~'\\s'
assert 'aaa'=~'\\S'

def datePattern = '([0-9]{4})-([0-9]{1,2})-([0-9]{1,2})'
def date = '2015-06-28'
def matcher = date=~datePattern
matcher.matches()
println matcher[0]
